# vue 实例

## 数据与方法

Vue 实例被创建时,`data`对象中的所有属性加入到 Vue 的响应式系统中.

```javascript
var data = { a: 1 };

var vm = new Vue({
  data: data
});

vm.a === data.a; // true

vm.a = 2;
data.a; // 2

data.a = 3;
vm.a; // 3
```

**例外**: 使用`Object.freeze()`,会阻止修改现有的属性,同时相应系统无法追踪变化.

```javascript
var obj = {
  foo: "bar"
};

Object.freeze(obj);

new Vue({
  el: "#app",
  data: obj
});
```

```html
<div id="app">
  <p>{{foo}}</p>
  <!-- 这里点击 'foo' 不会更新 -->
  <button @click="foo = 'baz'">change it</button>
</div>
```

除了数据属性,vue实例还暴露了一些有用的实例属性与方法.它们都有前缀`$`,以便与用户定义的属性区分开来.
